# -Sistema-de-Inteligencia-de-Negocios-para-el-An-lisis-del-Gasto-P-blico-Universitario
Proyecto de Business Intelligence para analizar la ejecución del gasto público universitario del MEF. Incluye procesos ETL y modelo dimensional en SQL Server, y un dashboard interactivo en Power BI con KPIs como PIM, Devengado y % de avance para apoyar la toma de decisiones.
